Kalori is an AI-first calorie and nutrition tracker implemented as a Next.js 16 / React 19 TypeScript PWA, using Supabase, Gemini-backed server Route Handlers, Zod validation, Sentry, Serwist/service-worker offline support, and pnpm. Test tooling is Vitest for unit/integration coverage, Playwright for E2E/a11y, Lighthouse CI for performance, plus TypeScript and ESLint checks.

Recent commits show active late-MVP stabilization and bug-bundle work: the latest commit fixed Gemini vision and mobile manual fallback, the prior bundle addressed dashboard food data, duplicate logs, servings, notes, and image upload, and earlier work added library quotas, progress chart dates, birthday migration coverage, and deployment changelog updates. This points to Phase 5 polish/PWA plus production-readiness hardening across logging, library, dashboard, and progress flows.
