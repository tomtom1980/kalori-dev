Project slug: kalori

Kalori is an AI-first calorie and nutrition tracker PWA: a dark-only, single-user MVP built around authenticated onboarding, text/photo/library food logging, AI nutrition parsing, personal food library growth, dashboard ledger UI, progress views, and offline/PWA hardening. The stack is Next.js 16 with React 19, TypeScript, Supabase auth/storage/database, Gemini-backed server route handlers, Tailwind CSS 4, Radix UI, Zustand, Zod, Sentry, Serwist, and IndexedDB outbox support. Package manager is pnpm 10.29.3 on Node >=20.19.0. Test runner is Vitest for unit/integration tests (`pnpm test`, `test:unit`, coverage) plus Playwright for E2E/a11y and Lighthouse CI/bundle/schema checks. Recent work is late Phase 5 quality and PWA polish: IDB outbox, service-worker/offline work, schema/test expectation refreshes, settings birthday formatting, and a 2026-05-18 bug bundle around food logging, progress, and image analysis limits. `AGENTS.md` is missing at project root; planning docs are present.
